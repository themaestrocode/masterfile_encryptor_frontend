Usage Guide

Master File Encryptor is an easy to use encryption tool that gets the job done with just a few clicks.

File Encryption

To encrypt a file, select the file and provide an encryption key to secure access to the file. The encryption key is very important, so we advice you keep it save for decryption purpose. Next, click the "Encrypt file" button and a download button will be generated for you to download the encrypted file. Again, we do not save any encryption key in our system; this is to say that you are in full control of the content of your file.

File Decryption

To decrypt a file, simply select an already encrypted file and provide the key with which the file was encrypted. Likewise, click the "Decrypt file" button and a download button will be generated for you to download the decrypted file.

Choosing a MasterFile Encryption Mode

The MasterFile Enryption Modes are custom encryption modes for efficient and seamless encryption of files and easy file decryption to its original format and file type.
If you choose the Dynamic Encryption Mode, during decryption, the original file type and format will be automatically detected and attached back to your file incase there has been any changes since you encrypted it. However, using this mode results in a slightly higher file size compared to the Fixed mode because it uses Base64 encoding. If you choose the Fixed Encryption Mode the original file type and format cannot be automatically detected during decryption. This mode does not use Base64 encoding, and as a result, the size of the resulting file is lesser compared to the Dynamic mode.

As an example, if you encrypt a PDF file and you at some point changed the ".pdf" extension to ".jpg" for any reason, Dynamic mode will fix it back for you during decryption, but Fixed mode cannot.
